# 📄 Syafiqah Nazir's E-Resume & Portfolio Website

Welcome to my personal e-resume repository! This is a modern, responsive multi-page portfolio website designed to showcase my academic qualifications as an Information Management System graduate, professional skill sets, and project milestones.

🔗 **Live Link:** [View Live E-Resume Site](https://2025479318-dev.github.io/syafiqah-e-resume/)

---

## 🚀 Features

* **Multi-Page Architecture:** Divided into clear, dedicated sections for focused navigation (Home, About, Education, Skills, Experience, Achievements, Gallery, Contact, and Student view).
* **Interactive Components:** Features custom JavaScript handling interactive elements like smooth image galleries and animations.
* **Modern Typography & Icons:** Beautifully rendered using Google Fonts (*Playfair Display* & *Poppins*) and icons from Font Awesome v6.5.1.
* **Responsive Layouts:** Built using modern CSS techniques (Flexbox and Grid configurations) ensuring mobile and desktop responsiveness.

---

## 🛠️ Built With

* **HTML5:** Semantic structuring across pages.
* **CSS3:** Custom stylesheet system (`css/style.css`) incorporating dynamic transitions and grid configurations.
* **JavaScript:** Lightweight vanilla JS scripting to manage UI updates and enhance user interaction.
